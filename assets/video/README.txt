Realistic “rider arrives → shoots” intro (optional)
===================================================

This site can play a short FILMED clip as the opening (real person, bike, camera).
Code animation (SVG/CSS) cannot look truly realistic for full human motion.

Add either or both (browser picks the first it can play):

  • intro.webm
  • intro.mp4

Place them in this folder: assets/video/

Requirements for the clip:
  • A few seconds is enough (keep file size small for the web).
  • Muted playback (browsers require mute for autoplay).
  • 1920×1080 or 1280×720 is fine; it displays full-screen with object-fit: cover.

Optional poster (first frame while loading):

  • intro-poster.jpg

If no video file is present, the built-in SVG + polaroid intro runs instead.
